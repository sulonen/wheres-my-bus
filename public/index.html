WHERE'S MY BUS?

Code Fellows 301 Team Project

We aim to provide a short UX path from a King County Metro bus stop location to a visual representation of the current location of an inbound vehicle.

We are using the OneBusAway API in conjunction with Google Maps API to access and manipulate real-time data.
