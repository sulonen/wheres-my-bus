Where's My Bus?

Code Fellows 301 Team Project

Our aim is to provide a shorter UX path from King County Metro bus stop location to a visual representation of the location of an arriving vehicle.

We are using the OneBusAway API in conjunction with Google Maps API to access and manipulate real-time data.
